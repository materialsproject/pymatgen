pymatgen.analysis.diffraction.tem moduleÔÉÅ

This module implements a TEM pattern calculator.

class TEMCalculator(symprec: float | None = None, voltage: float = 200, beam_direction: tuple[int, int, int] = (0, 0, 1), camera_length: int = 160, debye_waller_factors: dict[str, float] | None = None, cs: float = 1)[source]ÔÉÅ

Bases: AbstractDiffractionPatternCalculator

Computes the TEM pattern of a crystal structure for multiple Laue zones. Code partially inspired from XRD calculation implementation. X-ray factor to electron factor

conversion based on the International Table of Crystallography.

Parameters:

  • symprec (float) ‚Äì Symmetry precision for structure refinement. If set to 0, no refinement is done. Otherwise, refinement is performed using spglib with provided precision.

  • voltage (float) ‚Äì The wavelength is a function of the TEM microscope‚Äôs voltage. By default, set to 200 kV. Units in kV.

  • beam_direction (tuple) ‚Äì The direction of the electron beam fired onto the sample. By default, set to [0,0,1], which corresponds to the normal direction of the sample plane.

  • camera_length (int) ‚Äì The distance from the sample to the projected diffraction pattern. By default, set to 160 cm. Units in cm.

  • symbol (debye_waller_factors ({element) ‚Äì float}): Allows the specification of Debye-Waller factors. Note that these factors are temperature dependent.

  • cs (float) ‚Äì the chromatic aberration coefficient. set by default to 1 mm.
