This module provides various representations of transformed structures. A TransformedStructure is a structure that has been modified by undergoing a series of transformations.

  • return_alternatives ‚Äì Whether to return alternative TransformedStructures for one-to-many transformations. return_alternatives can be a number, which stipulates the total number of structures to return.

  • clear_redo ‚Äì Boolean indicating whether to clear the redo list. By default, this is True, meaning any appends clears the history of undoing. However, when using append_transformation to do a redo, the redo list should not be cleared to allow multiple redos.

Generates TransformedStructure from a cif string.

Parameters:

  • cif_string (str) ‚Äì Input cif string. Should contain only one structure. For CIFs containing multiple structures, please use CifTransmuter.

  • transformations (list[Transformation]) ‚Äì Sequence of transformations to be applied to the input structure.

  • primitive (bool) ‚Äì Option to set if the primitive cell should be extracted. Defaults to True. However, there are certain instances where you might want to use a non-primitive cell, e.g., if you are trying to generate all possible orderings of partial removals or order a disordered structure.

  • occupancy_tolerance (float) ‚Äì If total occupancy of a site is between 1 and occupancy_tolerance, the occupancies will be scaled down to 1.
